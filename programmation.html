Description

Dans ce module, j’ai appris à développer des scripts automatisés pour traiter les données et à interagir avec des API pour collecter des informations en temps réel. Ces compétences ont été appliquées à des projets concrets, comme l’intégration de bases de données météorologiques.

    Outils utilisés :
  • Python : Pour collecter, nettoyer et analyser les données via des scripts personnalisés.
  • API SYNOP : Pour extraire des données météorologiques en temps réel.
  • R : Pour manipuler les données et produire des graphiques interactifs.

Exemples de Projets Réalisés

Gestion des stations vélo (Vélib') :

Extraction et visualisation des données des stations Vélib' pour analyser leur disponibilité en temps réel. Le projet utilise des appels API pour mettre à jour les données dynamiquement.

Nettoyage de données :

Automatisation de la suppression des doublons et de la gestion des valeurs manquantes dans une base de données clients.
