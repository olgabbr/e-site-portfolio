Description

Dans ce module, j'ai appris à rendre mes analyses accessibles grâce à des visualisations interactives et dynamiques. Ce travail inclut la création de tableaux de bord pour explorer les données et d’outils graphiques pour appuyer les prédictions issues des analyses statistiques.:

    Outils utilisés :
  • Tableau : Conception de tableaux de bord interactifs pour synthétiser les données.
  • Power BI : Création de graphiques pour des analyses dynamiques.
  • ggplot2 (R Studio) : Réalisation de graphiques sur mesure pour représenter des données complexes.

Exemples de Projets Réalisés

Visualisation des données météo (SAE SID)

Création de graphiques et de cartes interactives pour représenter les données météorologiques récupérées via l’API SYNOP. Ce travail met en avant les tendances locales et régionales

Datavisualisation BUT 1
